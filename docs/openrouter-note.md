# OpenRouter & gateways note / OpenRouter 与兼容网关说明

This Termux port only adds Android compatibility; it does not change Codex behavior. For OpenRouter or other gateways, configure the model provider settings as you would on any Codex installation.

Key points:
- Use the standard Codex model provider configuration
- Set appropriate base URLs and API keys in your config
- Refer to provider-specific documentation for model names and formats

See [quickstart.md](./quickstart.md) for OpenRouter setup examples.
