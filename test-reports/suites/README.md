# Test Suites

This folder contains the runnable checklists used to generate the reports under
`test-reports/`.

Suites are split by release line and platform:

- LTS
- `lts/linux.md`
- `lts/termux.md`
- `lts/macos.md`
- Latest (Termux-only)
- `latest/termux.md`

The legacy universal checklist is preserved under `legacy/`.
